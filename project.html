Projects

Sempahores

A locking protocol mechanism to solve the reader/writers problem was implemented with priority inheritance to solve the priority inversion between processes with different priorities for the XINU OS.

Scheduler

For improving the scheduling of processes in the XINU OS, designed two scheduling implementations - random (fair-policy) and Linux (goodness measure) scheduler.

Demand paging

Implemented demand paging for XINU OS. Also designed a FCFS and Second Chance page replacement policy for swapping pages in and out of backing stores to physical memory.

Queue simulation

Successfully simulated queuing behavior of 30000 odd packets over single server queue to gather statictics which can be used to ensure efficient bandwidth by considering the end to end daly of these packets over five queues.

MPLS Simulation

Using GNS3 simulator carried out different kind(ATM, ATOM, etc) of traffics over MPLS network on Cisco routers

Jboss Framework-Working with business processes and enterprise service bus

Implemented a travel booking system where air-service, hotel-service, and ticket-service communicated with each other using the Jboss architecture. Also dependencies were implemented(for e.g. if air service and hotel succeed then fail the booking service etc.) for the three service

Gmail visualization

Using a servlets, JSP and JavaScript charting libraries analysis of inbox of a Gmail account implemented. Java mail API used to interact with Gmail Server. Once connection is established analysis is carried out from various perspectives to produce interesting charts and analysis about the user email activity.

User level and kernel level exploits - Cent OS

Written two exploits - one at user level which exploits the strcpy() function of C library and gains root access by buffer overflow attack. Second which loads a kernel module and does a kernel level overflow using unsafe kernel level function thereby gaining access of system at kernel level.

Open source projects

Worked on three open source project related to expertiza system used by NC-State university Professor Ed Gehringer. Implemented logging functionality for expertiza and implemented a module to let the user advertise for partners for course assignement within expertiza. The coding was done using Ruby on Rails - MVC architecture.

LUMINA- Laser Aided Mouse Controller

To increase the mobility of the presented implemented C# project which simulates mouse movements from laser projected on wall. The camera tracks the Laser movements and then the algorithm detects the brightest point from the start till the end of the movement and correspondingly simulates mouse cursor movements. Also a back propagation neural network was implemented to recognize the mouse events like click, scroll etc from the gestures made from the laser

MANET for Android networks

Designed a ad-hoc routing protocol for Android based phones. The routing algorithm was tested on a custom developed chat application which could trace the route from source to destination and back

Early warning intrusion detection system using HoneyD Honeypot

Submitted a paper for early warning intrusion detection system designed usng honeyd farmework which saves deployement costs and can implement thousands of IP's using a single vitual honeypot, instead of using a conventional honeypot framework.
